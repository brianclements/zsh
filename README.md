# My Zsh Folder

Packaged with [dotfiles][1].

[1]: https://github.com/brianclements/dotfiles.git

## Installation

* Install to dotfiles location
* symlink the `.zsh` folder itself to `./`
* symlink `zshenv` and `zshrc` to `~/` as `.zshenv` and `.zshrc`
